hammer2-fuse ([v0.4.2](https://github.com/kusumi/hammer2-fuse/releases/tag/v0.4.2))
========

## About

HAMMER2 userspace file system in Rust

## Supported platforms

Linux / FreeBSD

## Requirements

Rust 1.87.0 or newer

## Build

    $ make

## Install

    $ make install

## Uninstall

    $ make uninstall
