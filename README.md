hammer2-fuse ([v0.5.0](https://github.com/kusumi/hammer2-fuse/releases/tag/v0.5.0))
========

## About

HAMMER2 userspace file system in Rust

## Supported platforms

Linux / FreeBSD

## Requirements

Rust 1.96.0 or newer

## Build

    $ make

## Install

    $ make install

## Uninstall

    $ make uninstall
